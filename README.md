# illusory0x0/posix

## Installation

### Install from mooncake

```bash
moon add illusory0x0/native
bash .mooncakes/illusory0x0/native/install.sh

moon add illusory0x0/posix
bash .mooncakes/illusory0x0/posix/install.sh
```
